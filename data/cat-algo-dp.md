## 1. 什么是动态规划？核心要素？

> **难度**: medium | **分类**: 算法-动态规划 | **ID**: 87

#### 🎯 本质
动态规划（Dynamic Programming，DP）是将**大问题拆成重叠子问题**，通过**存储子问题的解**避免重复计算，从而高效求解的算法思想。核心是用**空间换时间**。

#### 🧒 类比
像考试时做笔记——算过的题目把答案记下来，下次遇到直接查，不用重新算。比如斐波那契数列，f(5)需要f(4)和f(3)，而f(4)又需要f(3)和f(2)，f(3)被重复计算了——记下来就不重复算了。

#### 📊 图解
动态规划两要素:
① 最优子结构:
  大问题的最优解由子问题的最优解组成
  例: 最短路径=子路径最短之和

② 重叠子问题:
  子问题会被反复计算
  例: fib(5)中fib(3)计算了2次

解题四步法:
1. 定义dp数组含义(dp[i]代表什么)
2. 找状态转移方程(递推关系)
3. 确定初始值和边界条件
4. 确定遍历顺序(从前往后/从后往前)

斐波那契示例:
  dp[i] = dp[i-1] + dp[i-2]
  dp[0]=0, dp[1]=1
  fib(5) = 0,1,1,2,3,5
#### 🔧 详解
**自顶向下（记忆化搜索）**：用递归+缓存（备忘录），按需计算。**自底向上（递推）**：从初始值开始逐步推导到目标值，通常用循环实现。自底向上更常见，避免了递归栈开销。**空间优化**：如果dp[i]只依赖前几个状态，可以用滚动变量代替整个数组（如斐波那契只需两个变量）。

动态规划和贪心的区别？答：贪心只看当前最优（局部最优），不回头；DP考虑所有子问题的最优组合（全局最优）。动态规划和分治的区别？答：分治的子问题不重叠（如归并排序），DP的子问题有重叠（如斐波那契）。

## 2. 背包问题？

> **难度**: medium | **分类**: 算法-动态规划 | **ID**: 88

#### 🎯 本质
背包问题是动态规划的经典应用：给定容量为W的背包和N个物品（各有重量和价值），求**能装入背包的最大价值**。分为0-1背包（每个物品最多选1次）和完全背包（每个物品可选多次）。

#### 🧒 类比
像出国旅行装箱——行李箱有重量限制，每种物品有重量和价值。0-1背包像只能带一件衣服，完全背包像同一个T恤可以带多件。目标是让行李箱里的东西总价值最高。

#### 📊 图解
0-1背包(每个物品选或不选):
  dp[i][w] = 前i个物品,容量w的最大价值
  不选i: dp[i-1][w]
  选i:   dp[i-1][w-weight[i]] + value[i]
  dp[i][w] = max(不选, 选)

完全背包(每个物品可选多次):
  dp[i][w] = max(dp[i-1][w], dp[i][w-weight[i]] + value[i])
  注意: 选i时是dp[i]不是dp[i-1]
  因为第i个物品还可以再选!

空间优化(一维数组):
  0-1背包: 从后往前遍历w
    (防止同一物品被选多次)
  完全背包: 从前往后遍历w
    (允许同一物品被选多次)

初始化: dp[0][w] = 0 (没有物品时价值为0)
时间复杂度: O(N*W)
#### 🔧 详解
**0-1背包**的关键：二维dp中第i行只依赖第i-1行，因此可以压缩为一维。从后往前遍历是为了防止当前物品被重复选取。**完全背包**从前往后遍历，因为允许重复选。背包问题变体很多：多重背包（每种物品有限数量）、分组背包、混合背包等。

## 5. 背包问题？

> **难度**: medium | **分类**: 算法-动态规划 | **ID**: 407

#### 🎯 本质
0-1背包：**dp[i][w]=max(dp[i-1][w],dp[i-1][w-weight[i]]+value[i])**。每个物品选或不选。

#### 🧒 类比
背包问题像整理行李箱——每件物品(选/不选)让总价值最大不超过容量。

#### 📊 图解
dp[i][w]: 前i个物品容量w的最大价值
for each item:
  for w=W...weight[i]:
    dp[w]=max(dp[w], dp[w-weight]+value)
#### 🔧 详解
0-1背包每个物品只能选一次。完全背包物品可以选多次(内层循环正序)。分组背包每组只能选一个。注意0-1背包内层逆序遍历。
#### 💻 代码
function knapSack(W,weight,value){
  const dp=new Array(W+1).fill(0);
  for(let i=0;i=weight[i];w--)
      dp[w]=Math.max(dp[w],dp[w-weight[i]]+value[i]);
  return dp[W];
}
#### ❓ 追问
0-1背包为什么内层逆序？答：保证每个物品只选一次。完全背包为什么正序？答：允许重复选。

## 15. 股票买卖最佳时机？

> **难度**: medium | **分类**: 算法-动态规划 | **ID**: 417

#### 🎯 本质
买卖股票：**低买高卖求最大利润**。一次交易：维护最低价格。多次交易：贪心。

#### 🧒 类比
股票买卖像做差价——最低点买入最高点卖出。

#### 📊 图解
一次: maxProfit=max(maxProfit, price-minPrice)
多次: 每次涨就卖
#### 🔧 详解
一次交易：遍历记录最低价格，每天计算如果今天卖的利润。多次交易：每次上涨就买卖(贪心)。含冷冻期/手续费需要DP。
#### 💻 代码
// 一次交易
function maxProfit(prices){
  let minPrice=Infinity,maxProfit=0;
  for(const p of prices){
    minPrice=Math.min(minPrice,p);
    maxProfit=Math.max(maxProfit,p-minPrice);
  }
  return maxProfit;
}
// 多次交易(贪心)
function maxProfit2(prices){
  let profit=0;
  for(let i=1;iprices[i-1]) profit+=prices[i]-prices[i-1];
  return profit;
}
#### ❓ 追问
含冷冻期怎么做？答：三个状态dp：持有/不持有冷冻/不持有非冷冻。
